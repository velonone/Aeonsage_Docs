## Requirements

### Minimum Requirements

| Resource | Requirement |
|----------|-------------|
| **CPU** | 2 cores |
| **RAM** | 4 GB |
| **Storage** | 20 GB SSD |
| **Network** | 100 Mbps |
| **OS** | Ubuntu 20.04+, Debian 11+, or RHEL 8+ |

### Recommended for Production

| Resource | Requirement |
|----------|-------------|
| **CPU** | 4+ cores |
| **RAM** | 8+ GB |
| **Storage** | 100 GB SSD |
| **Network** | 1 Gbps |
| **Database** | PostgreSQL 15+ |

<Note type="warning" title="Local Models">
If running local AI models (Ollama):
- Add 8+ GB RAM per model
- GPU recommended for better performance
- Consider dedicated GPU instances
</Note>

## Security Considerations

<Accordion>
  <AccordionItem title="Network Security">
    
    - Use HTTPS/TLS for all external traffic
    - Restrict database access to Gateway instances only
    - Use VPN or private networking for inter-service communication
    - Enable firewall rules to limit port exposure
    
  </AccordionItem>
  <AccordionItem title="Authentication">
    
    - Generate strong admin tokens (32+ characters)
    - Use environment variables for secrets
    - Rotate tokens periodically
    - Enable rate limiting on public endpoints
    
  </AccordionItem>
  <AccordionItem title="Data Protection">
    
    - Enable encryption at rest for databases
    - Regular backups with encrypted storage
    - Audit logging for compliance
    - Data retention policies
    
  </AccordionItem>
</Accordion>
